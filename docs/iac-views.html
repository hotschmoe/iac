VIEW ARCHITECTURE

Command Center is the default — where you manage your economy, queues, and fleets at a strategic level. Most LLM agent interaction happens here (build, research, dispatch fleets with policies).

Windshield is entered per-fleet — you "drop into" a fleet and pilot it directly. The node graph shows only your current sector and immediate connections. Adjacent hostiles appear as warnings. Movement is directional (pick an exit). The policy system still runs but you override it with direct commands. This is the exploration experience — you're charting unknown space one hop at a time.

Star Map is an overlay opened from either view. Three zoom levels:
  Close — node graph, ~3 hex radius. Same style as windshield but scrollable.
  Sector — hybrid hex-cell, ~8 hex radius. Good for route planning.
  Region — minimal dots + lines, ~20 hex radius. Shows the big picture, fleet positions, homeworld.

Waypoints can only be set on sectors you've explored. The fleet auto-navigates along your discovered edges. If a route requires traversing unknown edges, no path is available — you must explore manually first via windshield.

Fog of war: ░░░ sectors are unexplored. They exist (generated from seed) but you don't know their contents until a fleet visits or a Sensor Array reveals them. The map only shows your personal explored territory.
